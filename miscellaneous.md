
### MySql Create a users and a database 
```
CREATE USER 'ex1'@'localhost' IDENTIFIED BY '***';
GRANT ALL PRIVILEGES ON *.* TO 'ex1'@'localhost' REQUIRE NONE WITH GRANT OPTION MAX_QUERIES_PER_HOUR 0 MAX_CONNECTIONS_PER_HOUR 0 MAX_UPDATES_PER_HOUR 0 MAX_USER_CONNECTIONS 0;
CREATE DATABASE IF NOT EXISTS `ex1`;
GRANT ALL PRIVILEGES ON `ex1`.* TO 'ex1'@'localhost';
```
